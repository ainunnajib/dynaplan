## Architecture
- Backend: FastAPI, 1645 tests, file-backed SQLite with NullPool for tests (+ optional PostgreSQL)
- Rust Engine: dynaplan-core crate with sparse block storage, formula engine, dependency graph, parallel calc (rayon), spread/aggregation, PyO3 bridge
- Formula Engine: 100+ functions across time, lookup, text, statistical, aggregation categories (Python + Rust parity)
- Frontend: Next.js 15 + TypeScript + Tailwind + Recharts, TS compiles clean
- Cell storage: sparse table with sorted pipe-separated dimension_key, optional at-rest encryption
- Security: RBAC, DCA, SSO (SAML/OIDC), SCIM, IP allowlisting, mTLS, per-key rate limiting, model encryption with KMS
- Data Integration: CSV/Excel, REST API, CloudWorks connectors (S3/DB/SFTP/HTTP), Data Hub staging, Anaplan Connect CLI
- Execution: background job executor, CloudWorks cron scheduler, pipeline runtime, ALM promotion engine
- Observability: Prometheus metrics, Grafana dashboards, health API, horizontal scaling control plane
- Deployment: Kubernetes manifests, Redis shared state, sticky sessions, HPA
